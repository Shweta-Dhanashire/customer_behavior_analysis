Customer Behavior Analysis :- 
Data Analytics project showcasing customer behavior analysis using Python, SQL, and Power BI.

Overview :- 
This project demonstrates an end-to-end data analytics workflow, including data loading, cleaning, exploratory data analysis (EDA), SQL querying, and dashboard creation. The objective is to analyze customer data and generate meaningful business insights.

Dataset :-
The dataset contains structured data related to customer transactions, products, and sales. It was used to analyze customer behavior and evaluate business performance.

Tools Used :-
Python (Pandas, NumPy, Matplotlib, Seaborn)
MySQL Server
Power BI
Jupyter Notebook
Gamma (for presentation)

Project Steps :- 
Loaded and cleaned data using Python
Performed Exploratory Data Analysis (EDA)
Executed SQL queries in MySQL for data analysis
Built an interactive dashboard in Power BI
Created a report and presentation

Dashboard :-
The Power BI dashboard presents key insights such as sales trends, customer analysis, and product performance using interactive visualizations.

Results :-
The analysis identified customer behavior patterns, top customers, and product performance. These insights help understand business trends and support data-driven decision-making.
